    Welcome to Open GOOP Development Suite

    This repository is for a fully open source version of GOOP Development Suite. This version is not a replacement of NI’s Version GOOP Development Suite, it’s only a test bed where we can try things a bit easier and faster. I recommend you to use NI’s version to start with: http://sine.ni.com/nips/cds/view/p/lang/en/nid/209038

    How to get started: You need to clone the Repository into your LabVIEW20XX folder (currently the Master branch is in LabVIEW2012). To do this you can clone it into a temporary directory and then copy this into the LabVIEW-folder. In Windows: 

    git clone https://github.com/opengds/OpenGDS.git c:\LabVIEW2012-temp 
xcopy C:\labview2012-temp C:\Program Files (x86)\National Instruments\LabVIEW 2012 /H /Y /R /S
