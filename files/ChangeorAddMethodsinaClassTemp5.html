Change or Add Methods in a Class Template

You can change the already existing methods templates or add new ones to any  existing class template. This is done by

  1. first adding a new template VI or modify an existing and then
  2. register the new template VI by updating the method.ini file.


Change or add new method template VI:s

  • Open and modify an already existing method from the Method folder in the selected class template or add a new one to this folder.
  • The class templates are placed in (for GOOP 3): …\National Instruments\LabVIEW xxxx\resource\Framework\Providers\Open_GDS\ClassProviders\Provider_EndevoGOOPxxx\Templates


Register the new methods

Open the methods.ini configuration file (you will find the configuration file under the Methods folder in the class folder).


The ini file looks like this example:


[NormalMethods]

Read Attribute=B_ReadAttribute.vi

Write Attribute=B_WriteAttribute.vi

HR1=

Read Class Attribute=B_ReadClassAttribute.vi

Read & Write Class Attribute=B_ReadWriteClassAttribute.vi

HR2=

None Modifying Method=B_NoneModifying.vi


[PropertyMethods]

Get Property=B_GetProperty.vi


Under [NormalMethods] you find de methods that you can select in the Create Method Dialog when you select Method Type “Normal”


Under [PropertyMethods] you find de methods that you can select in the Create Method Dialog when you select Method Type “Property Method”


On the left side of the “= “ is the name that will be shown in the Create Method Dialog.

On the right side of the “= “ is the name of the VI that will be connected to the name on the left side. The VI must be placed in the method folder.


HR? Is a delimiter between methods. Will be shown as a line in the “Type of Method to create” display in the Create Method Dialog.
