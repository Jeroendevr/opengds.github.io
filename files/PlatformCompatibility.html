Platform Compatibility


The GOOP Development Suite tool can be installed on Windows XP/Vista/7.


The tool functions will be the same for any application instance, for example MyComputer or RT. The debugger will not work for remote debugging. Read more about this in the debugger section.


The GOOP3 and GOOP4 code created by the tool will run on Windows and RT targets. The code should actually run on any target although we cannot provide support for targets other than Windows and RT.


Note. If you move GOOP code to another computer, make sure to bring the run time llb files from the development machine. These are:

…\vi.lib\addons\_goop3.llb (for GOOP 3)

…\vi.lib\addons\_goop4.llb (for GOOP 4)
